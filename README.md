# Survive_horder

Survive Horder é um jogo de plataforma de sobrevivência onde o objetivo é derrotar ondas de inimigos enquanto navega por plataformas flutuantes. O jogo aumenta a dificuldade conforme você avança de fase, exigindo agilidade e precisão nos ataques.

Controles
Seta Esquerda: Move o personagem para a esquerda.

Seta Direita: Move o personagem para a direita.

Seta Cima: Pula.

Tecla K: Ataca os inimigos (golpe de chute).

Como Executar:

Para rodar o jogo, certifique-se de ter o Python e a biblioteca pgzero instalados.

Abra o terminal (ou PowerShell) na pasta do projeto.

Certifique-se de que a estrutura de pastas (images/, sounds/ e main.py) esteja corretas.

Execute o comando: python main.py
